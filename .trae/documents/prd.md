## 1. 产品概述

企业现金流 AI 预测看板是一款面向财务经理的智能资金管理工具，通过 AI 驱动的预测模型提前判断回款、付款和资金缺口，帮助企业在资金链断裂前采取行动。
- 解决财务经理对现金流不确定性缺乏前瞻性判断的痛点，提供从数据接入到风险预警的全链路决策支持
- 目标用户为企业财务经理及 CFO，核心价值是将事后核算转变为事前预判

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 财务经理 | 管理员分配账号 | 全部功能：数据接入、预测查看、情景推演、预警管理 |
| 财务专员 | 管理员分配账号 | 数据接入、预测查看、添加备注 |

### 2.2 功能模块

1. **首页**：资金总览仪表盘、关键指标卡片、资金缺口趋势、预警摘要、快捷操作入口
2. **数据接入**：导入应收应付数据、识别异常记录、数据预览与校验
3. **预测**：按月份预测现金流、展示资金缺口、设置安全余额、锁定预测版本
4. **客户回款**：按客户预测回款概率、回款分布分析、客户风险评级
5. **供应商付款**：按供应商预测付款压力、付款优先级排序、付款日历
6. **情景推演**：模拟延迟回款、模拟提前采购、多方案对比、生成推演报告
7. **预警**：生成风险等级、推送预警、添加人工备注、锁定预测版本、对比实际结果、导出经营摘要

## 3. 核心流程

用户从数据接入开始，上传应收应付数据后系统自动识别异常。确认数据后进入预测模块，AI 生成月度现金流预测和资金缺口分析。用户可进一步按客户查看回款概率、按供应商查看付款压力。在情景推演中模拟不同经营策略的影响。最终通过预警模块获取风险提示、添加备注、对比实际结果并导出报告。

    "供应商付款预测" --> "情景推演"

    "锁定预测版本" --> "对比实际结果"

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深海军蓝 (#0B1D3A) + 冰蓝强调色 (#00D4FF) + 琥珀警告色 (#FFB020) + 珊瑚红危险色 (#FF4757)
- 按钮风格：圆角 (rounded-lg)，主按钮渐变背景，hover 时微上浮 + 阴影增强
- 字体：标题使用 DM Sans（Bold/Semibold），正文使用 DM Sans（Regular），数据数字使用 JetBrains Mono
- 布局风格：左侧固定导航栏 + 右侧内容区，卡片式模块布局
- 图标风格：线性图标（Lucide），配合颜色语义化

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 首页 | 资金总览卡片 | 4张统计卡片，深色玻璃质感，数字用 JetBrains Mono，带趋势箭头和迷你折线图 |
| 首页 | 现金流趋势图 | 大面积折线/面积图，实际线实色+预测线虚色，底部渐变填充 |
| 首页 | 资金缺口预警区 | 横向进度条+缺口标注，低于安全线变红 |
| 首页 | 预警摘要 | 右侧窄卡片，3条预警缩略，左侧彩色竖条标识风险等级 |
| 数据接入 | 数据导入 | 拖拽上传区域，虚线边框，支持文件图标动画 |
| 数据接入 | 异常识别 | 表格行内红色/橙色高亮，异常图标悬浮显示原因 |
| 预测 | 月度预测图 | 柱状图分乐观/中性/悲观三色，安全余额线为虚线 |
| 预测 | 资金缺口 | 倒三角缺口标记，红色填充区域 |
| 客户回款 | 回款概率 | 客户卡片列表，左边缘颜色条标识概率等级（绿/黄/红） |
| 供应商付款 | 付款压力 | 热力矩阵，深蓝→浅蓝→橙→红色阶 |
| 情景推演 | 滑块模拟 | 自定义滑块组件，拖动时图表实时动画更新 |
| 预警 | 风险等级 | 四色圆点指示器 + 预警卡片列表 |
| 预警 | 对比结果 | 双柱对比图（预测 vs 实际），偏差率百分比标注 |

### 4.3 响应式设计

- 桌面优先设计，1280px 以上完整展示
- 1024-1280px 紧凑布局，侧边栏可折叠
- 768px 以下侧边栏变为底部导航，卡片改为单列

### 4.4 动效设计

- 页面切换：淡入+微上移动画
- 数据加载：骨架屏 + 数字滚动计数器
- 图表绘制：从底部生长动画
- 卡片 hover：微上浮 + 阴影加深
- 预警出现：从右侧滑入通知
